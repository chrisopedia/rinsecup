# Rinse Cup: Clean Up

The marketing page for Dan McNab's new sanitary product for the tattoo industry.

## Technologies Used

### Front End
- (x)HTML(5)
- (s)CSS
- JS on jQuery

### Templating
- Jekyll

### Build
- Grunt
- Bower

## Installation

1. `bower install`
2. `npm install`
