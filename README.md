# Rinse Cup: Clean Up

[![Greenkeeper badge](https://badges.greenkeeper.io/chrisopedia/rinsecup.com.svg)](https://greenkeeper.io/)

The marketing page for Dan McNab's new sanitary product for the tattoo industry.

## Technologies Used

### Front End
- (x)HTML(5)
- (s)CSS
- JS on jQuery

### Templating
- Jekyll

### Build
- Grunt
- Bower

## Installation

1. `bower install`
2. `npm install`
